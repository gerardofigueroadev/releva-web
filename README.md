# releva-web

Panel web de Releva — dashboard para supervisores y administradores de empresa, construido con Next.js.

## Stack

- **Framework:** Next.js (React + TypeScript)
- **Estilos:** Tailwind CSS
- **Estado:** React Query

## Funcionalidades principales

- Dashboard con estado de proyectos en tiempo real
- Revisión y aprobación de levantamientos por supervisor
- Descarga del folio Excel listo para enviar al ente regulador
- Reportes por empresa
- Gestión de usuarios y roles

## Requisitos

- Node.js >= 20

## Inicio rápido

```bash
npm install
cp .env.local.example .env.local
npm run dev
```
